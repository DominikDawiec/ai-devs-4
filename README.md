Repozytorium do nauki kursu **AI Devs 4** — materiały, notatki i ćwiczenia z poszczególnych lekcji.

```
AI_Devs_4/
├── S01E01/   Programowanie interakcji z modelem językowym

├── S01E04/   Wsparcie multimodalności oraz załączników
├── S01E05/   Zarządzanie jawnymi oraz niejawnymi limitami modeli
├── S02E01/   Zarządzanie kontekstem w konwersacji
├── S02E02/   Zewnętrzny kontekst narzędzi i dokumentów
├── S02E03/   Dokumenty oraz pamięć długoterminowa jako narzędzia
├── S02E04/   Organizowanie kontekstu dla wielu wątków

```

Każdy katalog zawiera folder `LESSON/` z materiałami do lekcji.
