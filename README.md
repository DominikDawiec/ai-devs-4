# AI Devs 4

Moje rozwiązania zadań z kursu **[AI Devs 4](https://www.aidevs.pl/)**.

> Kurs jest prywatny, dlatego to repozytorium zawiera wyłącznie moje propozycje rozwiązań zadań. Materiały z lekcji (notatki, transkrypcje, pliki kursowe) są przechowywane lokalnie i wykluczone z repozytorium przez `.gitignore`.

## Struktura

```
S01/
├── E01/
│   ├── notes/      ← materiały z lekcji (lokalne, niewidoczne w repo)
│   └── solution/   ← moje rozwiązanie zadania
├── E02/
│   ├── notes/
│   └── solution/
...
S05/
└── E05/
    ├── notes/
    └── solution/
```
